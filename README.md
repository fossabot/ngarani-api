# Ngarani API

## Introduction

This is a simple API to get random names.

## Usage

- Endpoint: `/api/get-name`
- Method: `GET`

### Response Example
```json
{
  "name": "Random Name"
}

## Contribution
To add more names, follow these steps:

### 1. Fork the repository.
### 2. Clone your fork.
### 3. Add names to the names.json file.
### 4. Commit your changes.
### 5. Create a Pull Request.
